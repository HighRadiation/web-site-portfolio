# 2026-05-02 00:28:00 - Bento Grid Planı

## Yazılan Prompt
Sunulan tasarım önerileri arasından Bento Grid yerleşimi (1), Glassmorphism efektleri (4) ve Sidebar Explorer (5) konseptleri onaylanmış olup, uygulama öncesinde Bento Grid yapısı için bir geliştirme planı oluşturulması istenmiştir.

## Yapılan Plan
Projeler bölümünü standart 3 kolonlu yapıdan daha dinamik olan Bento Grid yapısına geçirmek için aşağıdaki adımlar izlenecektir:

1. **CSS Güncellemesi (`globals.css`):**
   - `.projects-grid` sınıfı 4 kolonlu bir yapıya (desktop için) geçirilecek.
   - Farklı kart boyutları için yardımcı sınıflar tanımlanacak:
     - `.project-card-large` (2x2): Öne çıkan ana proje.
     - `.project-card-wide` (2x1): Geniş projeler.
     - `.project-card-standard` (1x1): Standart projeler.
   - Row height (`grid-auto-rows`) belirlenerek kartların dikeyde hizalanması sağlanacak.

2. **Bileşen Mantığı (`ProjectsSectionClient.tsx`):**
   - Projeler map edilirken `index` değerine göre farklı CSS sınıfları atanacak.
   - Örnek dağılım: Index 0 (Large), Index 1-2 (Standard), Index 3 (Wide), vb.

3. **Responsive Düzenlemeler:**
   - Tablet boyutunda 2 kolonlu yapıya geçilecek.
   - Mobil boyutunda tek kolonlu standart yapı korunacak.

4. **Görsel İyileştirmeler:**
   - Kartların içindeki "tarayıcı önizleme" (noktalar ve başlık) kısmının farklı boyutlarda düzgün görünmesi sağlanacak.

## Yapılan İşlem
- Uygulama planı oluşturuldu ve bu döküman hazırlandı.
- Bir sonraki adımda CSS ve React bileşenlerinde değişiklikler yapılacaktır.
