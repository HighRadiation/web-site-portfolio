# ESLint + Prettier Kurulumu ve Tüm Lint Hatalarının Temizlenmesi

## Yazılan Prompt
- Parantez stili değişikliği denendi ve geri alındı.
- ESLint + Prettier kurulumu istendi (önce Airbnb, sonra sadeleştirildi).
- Tüm dosyalardaki eksik return type'lar eklenmesi istendi.
- React 19 JSX namespace hatası düzeltilmesi istendi.

## Yapılan Plan
1. Airbnb config'i kaldır, sade `eslint:recommended` + `@typescript-eslint/recommended` kullan.
2. `.eslintrc.json` ve `.prettierrc` dosyalarını oluştur.
3. `package.json` script'lerine `ESLINT_USE_FLAT_CONFIG=false` ekle (ESLint 9 uyumu).
4. 19 dosyadaki tüm fonksiyonlara eksik return type ekle.
5. max-len (>100 karakter) ihlallerini düzelt.
6. unused-vars hatalarını düzelt (`_` prefix kuralı ekle).
7. React 19 için `JSX.Element` → `React.JSX.Element` güncelle.

## Yapılan İşlem

- **Eklenen:** `@typescript-eslint/parser`, `@typescript-eslint/eslint-plugin`, `prettier`, `eslint-import-resolver-typescript`, `prettier-plugin-organize-imports`

### Yeni Dosyalar

- `.prettierrc` — Prettier yapılandırması (singleQuote, trailingComma, printWidth: 100)
