## Yazılan Prompt

> 11. Admin UI'ı inline style'dan CSS'e taşı

## Yapılan Plan

Admin paneli altındaki tüm sayfalar (`layout.tsx`, `page.tsx` (Dashboard), `skills/page.tsx`, `skills/new/page.tsx`, `projects/page.tsx`, `projects/new/page.tsx`, `timeline/page.tsx`, `timeline/new/page.tsx`, `messages/page.tsx`) ile giriş sayfası (`login/page.tsx`) incelendi. 

**Tespit Edilen UI Yapısı ve Problemler:**
1. Sayfalardaki neredeyse tüm yapısal düzenler, flexbox/grid ayarları, arka planlar, butonlar, girdiler (inputs) ve kart bileşenleri inline style (`style={{...}}`) blokları ile doğrudan JSX içerisine gömülü durumdaydı.
2. Bu durum hem kodun okunabilirliğini ciddi şekilde düşürüyor hem de projede tutarlı bir admin teması uygulanmasını engelliyordu. Ayrıca ESLint kuralları gereği dosya satır uzunluklarında taşmalara (max-len) yol açıyordu.

**Strateji:**
- `src/app/globals.css` dosyasına admin paneline özel, premium karanlık mod tasarımıyla uyumlu ortak CSS sınıfları (layout, sidebar, stat kartları, form grupları, inputlar, butonlar, hata alertleri ve listeleme kartları vb.) eklenecek.
- Inline stillere sahip tüm admin sayfaları taranacak; inline stiller temizlenerek ilgili yeni CSS sınıfları (`className="..."`) ile değiştirilecek.
- Giriş sayfası (`login/page.tsx`) üzerindeki inline hata ve girdi stilleri de genel kurallara uygun olarak CSS sınıflarına dönüştürülecek.
- Değişikliklerden sonra `npm run lint` ve `npm run build` komutları çalıştırılarak maksimum satır uzunluğu (max-len) başta olmak üzere hiçbir derleme/lint hatası kalmadığından emin olunacak.

## Yapılan İşlem

### Değiştirilen Dosyalar

**`src/app/globals.css`**
- Dosyanın sonuna geniş kapsamlı `/* Admin Dashboard */` stilleri eklendi.
- `.admin-layout`, `.admin-sidebar`, `.admin-sidebar-brand`, `.admin-sidebar-nav`, `.admin-nav-link`, `.admin-logout-btn`, `.admin-main`, `.admin-content-wrapper` gibi yerleşim sınıfları tanımlandı.
- `.admin-stats-grid` ve `.admin-stat-card` gibi metrik kart stilleri oluşturuldu.
- `.admin-form`, `.admin-form-group`, `.admin-label`, `.admin-input`, `.admin-textarea`, `.admin-btn-primary`, `.admin-btn-outline` gibi form ve buton sınıfları eklendi.
- `.admin-card-list`, `.admin-card-item`, `.admin-card-unread`, `.admin-badge-new`, `.admin-message-body`, `.admin-empty-state` gibi listeleme elemanları tanımlandı.

**`src/app/[locale]/admin/layout.tsx`**
- Sidebar, navigasyon bağlantıları ve logout butonu inline stillerden arındırılarak CSS sınıflarına taşındı.

**`src/app/[locale]/admin/page.tsx` (Dashboard)**
- İstatistik metrik kartları ve genel layout yapısı tamamen CSS sınıflarına taşındı.
